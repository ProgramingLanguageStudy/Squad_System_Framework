### 기술 메모

- **Cinemachine 회전 저장/복원**: UI 열릴 때 CinemachinePanTilt의 PanAxis/TiltAxis Value 저장, 닫을 때 복원. UI 열려 있는 동안 InputAxisController 비활성화가 필수(입력 차단). 복원 후 N프레임 동안 컨트롤러 비활성화하여 복원값 덮어쓰기 방지.

- **끼임 탈출 패턴**: SettingsView 이벤트 → PlayScene 구독 → SquadController API. View는 UI만 담당, PlayScene이 Orchestrator 역할. SquadController.TeleportToDefaultPoint는 _spawnPoint 재사용으로 별도 Transform 불필요.

- **대화 끝내기 2단계**: TrySkipTyping 먼저 호출, true면 스킵만 하고 return. false면 EndDialogue. "다음" 버튼과 동일한 패턴 적용.

- **Character 상태 IsComplete**: AttackState 등이 IsComplete로 종료 신호. StateMachine이 주기적으로 확인해 Idle 전환. RequestIdle에서 Attack 상태일 때 early return으로 즉시 Idle 전환 방지.

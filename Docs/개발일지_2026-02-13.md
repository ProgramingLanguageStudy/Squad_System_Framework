### 기술 메모 / 배운 점

- **캐릭터 루트 vs 모델 스케일**: 로직·물리(CC, Collider, Rigidbody, 스크립트)는 빈 부모(스케일 1)에 두고, 모델·Animator만 자식에 두고 자식에서만 스케일 조정하면 이동·물리가 꼬이지 않음. 발을 Y=0(원점)으로 두면 배치·지면 체크가 단순함.
- **BlendShape 표정**: SkinnedMeshRenderer의 `SetBlendShapeWeight(인덱스, 0~100)`로 런타임에 제어. 이름은 `sharedMesh.GetBlendShapeIndex("Blink")` 등으로 조회. PicoChan은 face_base에 Blink, mouth_A~O, Joy, Angry, Sorrow, Fun 등 10개.
- **Git 긴 경로**: Windows에서 경로가 260자 넘으면 add 실패할 수 있음. `git config core.longpaths true`로 해결.

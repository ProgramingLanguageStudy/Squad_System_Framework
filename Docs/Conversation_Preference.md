# 대화 시 선호 사항 (AI 응답 방식)

프로젝트에서 AI와 대화할 때 적용할 규칙을 정리한 문서입니다.

---

## 1. 코드 수정/작성 여부

- **"코드 수정해줘", "스크립트 작성해줘"** 처럼 **명시적으로 요청한 경우**  
  → 실제 코드 수정·파일 작성 수행

- **그 외** (의견 물어보기, 어떻게 할지 궁금함, 좋을까/나쁠까 등)  
  → **코드 수정/작성 없이** 의견·생각·설명만 답변  
  → "이건 예시야, 바꿔달란 게 아니야" 같은 말이 있으면 코드 변경하지 않음

---

## 2. 정리 형태

- 의견·설명·정리 내용은 **가능하면 `.md` 파일로 정리**해 두기  
- `Docs/` 폴더 등에 두어서 나중에 참고용으로 사용

---

# 📅 개발일지

## 2025-02-10 (화요일)

### 오늘 한 일
- **인벤토리 디버거** 리스트 구조로 변경: (ItemData + 수량) 엔트리를 여러 개 등록하고, 각 행마다 아이템 데이터·수량·「추가」 버튼을 한 줄로 배치. 아이템/수량은 에디터에서 미리 등록, 적용(추가)만 플레이 모드에서 동작.
- **전체 디버거(Inventory, Player, EnemySpawner)** Find 로직 제거. 참조만 사용하고, 비어 있으면 로그로 「반드시 인스펙터에서 할당하세요」 안내.
- **플레이어 디버거**에 현재 스탯 표시 추가(체력, 사망 여부, 이동 속도, 공격력, 공격 속도, 방어력). 플레이 모드에서 0.1초 간격 Repaint로 실시간 갱신, 버프 물약 등 테스트용.
- **개발일지** 노션에 넣기 좋은 마크다운 템플릿 작성 (`Docs/개발일지_템플릿.md`).

### 기술 메모 / 배운 점
- Unity 커스텀 인스펙터에서 “실시간”처럼 값이 바뀌어 보이려면 **Repaint**가 필요함. 필드 참조만으로는 그리는 순간의 값은 맞지만, 화면이 다시 그려지지 않으면 예전 값으로 보임. Repaint는 0.1초 간격 등으로 제한하면 성능 부담을 줄일 수 있음.
